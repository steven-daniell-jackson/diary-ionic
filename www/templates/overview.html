Overview

This is a angular single page app that works with my node server for serving out my Dairy entries

Purpose: Easily add work that I have done daily

Reason: Doing it as my second Angular side project to get a better understanding of the MEAN stack and my memory is bad

Future plans: To save entries to Google sheets

Text Editor: Sublime Text 3

Node Packages: bcrypt-nodejs body-parser express gulp gulp-nodemon mongoose

Frameworks: Bootstrap AngularJS

Browser: Chrome

Operating System: Windows 10
